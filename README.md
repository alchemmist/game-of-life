A classic model to demonstrate the principles of evolution --- [Game of life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life). Conway's Game of Life is a cellular automaton devised by mathematician John Horton Conway. It consists of a grid of cells that evolve at each step according to a set of simple rules:
1. Each cell can be alive or dead.
2. A live cell with 2 or 3 live neighbors survives to the next generation.
3. A dead cell with exactly 3 live neighbors becomes alive (birth).
4. All other cells die or remain dead due to underpopulation or overpopulation.

Despite these simple rules, the game can produce surprisingly complex and beautiful patterns, making it a classic example of emergent behavior.


This repo is simple implementation of this game on Python usinnn PyGame. 

### Usage
After starting the play you can do these actions:
- `send the cells` with mouse click and mouse move
- `toggle evolution` by pressing the Space key
- `speed up time` by scrolling the mouse wheel
